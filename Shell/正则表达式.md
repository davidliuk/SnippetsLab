限定符表达

Note 表含义中的出现次数：限定符前面字符的出现次数。

| 限定符 | 表达含义                   |
| ------ | -------------------------- |
| *      | 出现次数>=0                |
| +      | 出现次数>=1                |
| ?      | 出现次数 0 or 1, 等价{0,1} |
| {n}    | 出现次数=n                 |
| {n,}   | 出现次数>=n                |
| {n, m} | n=< 出现次数<= m           |



定位符

| 定位符 | 表达含义                                                     |
| ------ | ------------------------------------------------------------ |
| ^      | 字符串开始的位置                                             |
| $      | 字符串结束的位置                                             |
| \b     | 限定单词(字)的字符，常用来确定一个单词，可以结合两个‘\b’使用 |
| \B     | 限定非单词(字)边界的字符，用的很少                           |
